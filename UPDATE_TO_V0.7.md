# Update RoundWise to v0.7

This update does not require a Supabase migration.

## Enhancements

- Visual tee-result location grid
- Automatic up-and-down calculation from chips/pitches and putts
- Color-coded Enter Point, Down Point, GIR, and 3+ putts in Details
- Triple circle for 3+ under and triple square for 3+ over
- Smaller home-screen improvement headline
- Cloud-sync warning moved to the bottom of Home
- Method score disabled in default metric settings for new users and Reset Defaults

Upload the project contents to the existing GitHub repository and commit to `main`. Cloudflare will deploy automatically.

Existing users with saved settings can use Settings → Reset defaults or turn Method score off manually.
